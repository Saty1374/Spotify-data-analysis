## 🎧 SPOTIFY DATA ANALYSIS DASHBOARD (POWER BI)

---

## 📌 PROJECT DESCRIPTION

This project focuses on analyzing Spotify music data using **Power BI** to create an interactive and visually rich dashboard.

The objective is to uncover insights related to song popularity, artist performance, and listening trends by transforming raw data into meaningful visualizations and KPIs.

The dashboard enables users to explore music trends, compare artists, and understand how different factors like album type, explicit content, and release time influence song popularity.

---

## 📂 DATASET OVERVIEW

- **Data Source:** Spotify Dataset  
- **Data Type:** Structured Music Data  
- **Content Includes:**  
  - Song Details  
  - Artist Information  
  - Popularity Metrics  
  - Album Types  
  - Duration & Release Data  

---

## 📊 DASHBOARD FEATURES

The Power BI dashboard provides the following key insights:

- **Total Songs & Artists Overview**  
- **Average Popularity & Duration Metrics**  
- **Top Artists based on Song Count**  
- **Top Songs based on Popularity**  
- **Songs Distribution by Album Type (Single / Album)**  
- **Explicit vs Non-Explicit Content Analysis**  
- **Year-wise Song Distribution**  
- **Monthly Popularity Trends**  
- **Monthly Song Release Patterns**  

---

## 📈 KEY KPIs

- **Total Distinct Songs**  
- **Total Artists**  
- **Average Popularity Score**  
- **Average Song Duration (seconds)**  

---

## 🔍 KEY INSIGHTS

- Identified the most popular songs and artists based on user engagement  
- Compared performance across different album types  
- Analyzed trends in song releases over months and years  
- Observed variation in popularity trends across time  
- Evaluated the distribution of explicit vs non-explicit content
